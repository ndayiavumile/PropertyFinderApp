#  PropertyFinder App

PropertyFinder is a mobile application that allows users to browse, search, and view property listings on an interactive map.  
It is designed to combine **Computer Science and GIS** to provide a smart property discovery experience.

---

##  Features

-  Interactive map-based property viewing  
-  Search and filter properties  
-  View detailed property information  
-  User authentication (Firebase)  
-  Data storage using Firestore  
-  Android-based mobile application  

---

##  Tech Stack

- Java (Android Development)
- Firebase Authentication
- Firebase Firestore
- Google Maps API
- Android Studio
